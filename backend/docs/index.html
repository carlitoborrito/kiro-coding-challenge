Quick Reference

GET /events

List all events. Supports ?status=active query parameter for filtering.

POST /events

Create a new event. Supports custom eventId or auto-generates one.

GET /events/{event_id}

Get a specific event by ID.

PUT /events/{event_id}

Update an existing event. Supports partial updates.

DELETE /events/{event_id}

Delete an event by ID.

GET /health

Health check endpoint.

Event Schema

Properties:
  • eventId (string) - Unique identifier
  • title (string) - Event title (1-200 chars)
  • description (string) - Event description (1-1000 chars)
  • date (string) - ISO format or YYYY-MM-DD
  • location (string) - Event location (1-200 chars)
  • capacity (integer) - Max attendees (1-100,000)
  • organizer (string) - Organizer name (1-100 chars)
  • status (enum) - active, scheduled, ongoing, completed, cancelled
